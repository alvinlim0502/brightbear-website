
# Bright Bear Preschool (Static Site)

Single-file static site to deploy with GitHub Pages.

## Deploy (GitHub Pages)
1. Create a new repository on GitHub (public).
2. Upload all files in this folder (including `assets/logo.png`).
3. Settings → Pages → Build and deployment: **Deploy from a branch** → Branch: **main** (root). Save.
4. Visit: `https://<your-username>.github.io/<repo-name>/`
